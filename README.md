# [待辦事項管理排程系統]

## Proposal Report

### 動機與目標
在日常生活中有許多的待辦事項，我們需要記住並根據每項的重要程度、期限來安排先後。因此做出這個工具紀錄各事項，並根據事項的特性來安排順序，減少這些瑣事帶來的勞心勞力。

### 預期功能
- 新增/刪除待辦事項（名稱、截止時間、優先級） 
- 顯示所有待辦事項  
- 標記事項為已完成  
- 根據優先級與截止時間自動排序  
- 顯示目前最需要優先完成的任務  

### 使用技術
- 程式語言：Python  
- 開發工具：VS Code  
- 資料儲存：JSON 或 TXT  
- 介面：CLI

### 時程規劃
| Week 9 | 設計資料結構（Class Task） |
| Week 10 | 完成新增、刪除、顯示功能 |
| Week 11 | 實作排序功能 |
| Week 12-15 | 根據review進行調整 |
